## 💻 Projeto

Em nosso projeto em grupo, iremos desenvolver um ambiente em realidade virtual, optando pela utilização da Unity para a implementação do VR. Além disso, para a modelagem de assets, personagens e ambientes, escolhemos integrar o Blender, buscando assim uma abordagem abrangente e eficiente. Nosso objetivo é criar uma experiência imersiva e interativa, explorando as potencialidades da realidade virtual no contexto escolhido.
